正定矩阵和半正定矩阵是线性代数和优化中的重要概念，它们具有一些非常有用的性质和结论，特别是在最优化问题中。让我详细解释一下它们的定义、性质以及这些性质如何在实际问题中被利用。

正定矩阵与半正定矩阵的定义

  • 正定矩阵：一个 n×nn \times nn×n 的实对称矩阵 AAA 被称为正定矩阵，如果对于所有非零向量 x∈Rnx \in \mathbb{R}^nx∈Rn，都有：

    xTAx>0x^T A x > 0 xTAx>0

    这意味着任何非零向量 xxx 乘以 AAA 后，再与 xxx 内积，结果总是正数。

  • 半正定矩阵：一个 n×nn \times nn×n 的实对称矩阵 AAA 被称为半正定矩阵，如果对于所有向量 x∈Rnx \in \mathbb{R}^nx∈Rn，都有：

    xTAx≥0x^T A x \geq 0 xTAx≥0

    这里允许 xTAxx^T A xxTAx 为零，但它不能为负数。

正定矩阵的性质

  1. 唯一性：如果 AAA 是正定的，那么线性方程组 Ax=bAx = bAx=b 有唯一解。这是因为正定矩阵总是可逆的（不存在零特征值），其逆矩阵也是正定的。

  2. 特征值：正定矩阵的所有特征值都是正数。半正定矩阵的所有特征值都是非负数。

  3. 矩阵的平方根：如果 AAA 是正定矩阵，那么存在一个唯一的正定矩阵 BBB 使得 B2=AB^2 = AB2=A，这个矩阵 BBB 称为 AAA 的平方根。

  4. 最优化中的应用：在二次型最小化问题中，如果二次型的系数矩阵是正定的，那么目标函数具有唯一的最小值。具体到问题中的损失函数 L(x)=12xTWTWx−xTWTb+constL(x) = \frac{1}{2} x^T W^T W x - x^T W^T b + \text{const}L(x)=21​xTWTWx−xTWTb+const，如果 WTWW^T WWTW 是正定的，那么该损失函数具有唯一的最小值。

半正定矩阵的性质

  1. 可能的多解性：如果 AAA 是半正定的，那么线性方程组 Ax=bAx = bAx=b 可能有无穷多解（当 AAA 不是满秩的情况）。

  2. 特征值：半正定矩阵的特征值是非负的，这意味着它的最小特征值是零或正数。

  3. 最优化中的应用：在最小化一个以半正定矩阵为系数的二次型函数时，如果矩阵是半正定但不是正定的，最小值可能不唯一，且可能出现在一个子空间上。

实际问题中的应用

在优化问题中，正定和半正定矩阵的性质决定了目标函数的形状和解的唯一性：

  • 正定矩阵的作用：在二次型最优化问题中，如 min⁡x12xTAx+bTx+c\min_x \frac{1}{2} x^T A x + b^T x + cminx​21​xTAx+bTx+c，如果 AAA 是正定矩阵，那么目标函数是一个严格凸函数，只有一个全局最小值。

  • 半正定矩阵的作用：如果 AAA 是半正定矩阵，那么目标函数是凸的，但可能存在平坦区域，这些区域会导致多个最小值，即解可能不唯一。

小结

  • 正定矩阵：确保唯一解和严格凸性。
  • 半正定矩阵：保证凸性，但解可能不唯一。

这些性质在优化问题、稳定性分析、机器学习（如支持向量机、线性回归）中广泛应用。正是因为这些性质，在线性方程组的解、优化问题的最小值以及特征值问题中，正定和半正定矩阵的概念非常重要。
