问题

如何将二次型 3x2+2xy+3y23x^2 + 2xy + 3y^23x2+2xy+3y2 通过正交变换化为只含 (x′)2(x')^2(x′)2 和 (y′)2(y')^2(y′)2 的标准形式.

推导

一个二次型可以表示为：

Q(x)=ax2+bxy+cy2=xTAx Q(\mathbf{x}) = ax^2 + bxy + cy^2 = \mathbf{x}^T A \mathbf{x} Q(x)=ax2+bxy+cy2=xTAx

其中，x\mathbf{x}x 是一个 222 维列向量，AAA 是一个对称矩阵

A=[ab/2b/2c] A= \begin{bmatrix} a & b/2 \\ b/2 & c \end{bmatrix} A=[ab/2​b/2c​]

由于 AAA 是对称矩阵，可以进行特征值分解：

A=PDP−1 A = P D P^{-1} A=PDP−1

其中，PPP 是 AAA 的特征向量矩阵，DDD 是由 AAA 的特征值构成的对角矩阵。

代入特征值分解

Q(x)=xTAx=xT(PDP−1)x Q(\mathbf{x}) = \mathbf{x}^T A \mathbf{x} = \mathbf{x}^T (P D P^{-1}) \mathbf{x} Q(x)=xTAx=xT(PDP−1)x

对于实对称矩阵 AAA，其特征向量是正交的。我们可以对这些特征向量进行归一化，使得 PPP 的列向量（特征向量）构成一个正交基，因此 PPP 是正交矩阵。正交矩阵满足:

PT=P−1 P^T = P^{-1} PT=P−1

进一步简化可以得到：

Q(x)=xT(PDPT)x=(PTx)TD(PTx) Q(\mathbf{x}) = \mathbf{x}^T (P D P^T) \mathbf{x} = (P^T \mathbf{x})^T D (P^T \mathbf{x}) Q(x)=xT(PDPT)x=(PTx)TD(PTx)

令 y=P−1x\mathbf{y} = P^{-1} \mathbf{x}y=P−1x，则 x=Py\mathbf{x} = P \mathbf{y}x=Py。代入后得到：

Q(x)=(Py)TA(Py)=yTPTAPy Q(\mathbf{x}) = (P \mathbf{y})^T A (P \mathbf{y}) = \mathbf{y}^T P^T A P \mathbf{y} Q(x)=(Py)TA(Py)=yTPTAPy

根据特征值分解，PTAP=DP^T A P = DPTAP=D，所以：

Q(x)=yTDy Q(\mathbf{x}) = \mathbf{y}^T D \mathbf{y} Q(x)=yTDy

结合上述变换过程，已知 PTAP=DP^T A P = DPTAP=D，我们得到：

Q(x)=yTDy Q(\mathbf{x}) = \mathbf{y}^T D \mathbf{y} Q(x)=yTDy

通过上述步骤，我们将一个二次型 Q(x)=xTAxQ(\mathbf{x}) = \mathbf{x}^T A \mathbf{x}Q(x)=xTAx 通过特征值分解和正交变换化为新坐标系y\mathbf{y}y下的标准形式 Q(x)=yTDyQ(\mathbf{x}) = \mathbf{y}^T D \mathbf{y}Q(x)=yTDy，其中 DDD 是对角矩阵。
