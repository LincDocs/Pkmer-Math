解析学・微積分

  1. RmR^mRm 上で微分可能な実数値関数 f(x) (x=(x1,x2,…,xm))f(x) \ (x = (x_1, x_2, \dots, x_m))f(x) (x=(x1​,x2​,…,xm​)) について, xi=vi(t) (i=1,2,…,m)x_i = v_i(t) \ (i = 1, 2, \dots, m)xi​=vi​(t) (i=1,2,…,m) とおく．ただし, 各 viv_ivi​ は RRR 上で微分可能な関数とする．次の各問いに答えよ.
    • (a) dfdt\frac{df}{dt}dtdf​ を ∂f∂xi\frac{\partial f}{\partial x_i}∂xi​∂f​ と dvidt (i=1,2,…,m)\frac{dv_i}{dt} \ (i = 1, 2, \dots, m)dtdvi​​ (i=1,2,…,m) で表せ.
    • (b) m=2,f(x)=x12+x1x2+2x22,v1(t)=sin⁡t,v2(t)=etm = 2, f(x) = x_1^2 + x_1 x_2 + 2x_2^2, v_1(t) = \sin t, v_2(t) = e^tm=2,f(x)=x12​+x1​x2​+2x22​,v1​(t)=sint,v2​(t)=et のとき, dfdt\frac{df}{dt}dtdf​ を求めよ.
  2. 次の微分方程式の一般解を求めよ. $$\frac{dy}{dx} - 2xy = e{x2}$$
  3. 閉曲線 CCC に沿った複素積分 $$\oint_C \frac{\cos z}{(2z - \pi)^3} , dz$$ を求めよ．ただし, CCC は円 ∣z∣=2|z| = 2∣z∣=2 とする.

以下逐题进行解析并给出答案。为方便阅读，记号中会将 ddx\frac{d}{dx}dxd​ 等常用符号直接写出，不再特别声明其含义。

1. 多元函数的全微分与链式法则

设 f:Rm→Rf: \mathbb{R}^m \to \mathbb{R}f:Rm→R 在 Rm\mathbb{R}^mRm 上可微，记

x=(x1,x2,…,xm),x = (x_1, x_2,\dots, x_m), x=(x1​,x2​,…,xm​),

并令每个分量 xi=vi(t)x_i = v_i(t)xi​=vi​(t)（其中 vi:R→Rv_i: \mathbb{R} \to \mathbb{R}vi​:R→R 可微）。

(a) dfdt\frac{df}{dt}dtdf​ 的表达式

由多元函数的链式法则可知

dfdt=∑i=1m∂f∂xi(x1,…,xm) dvi(t)dt.\frac{df}{dt} = \sum_{i=1}^m \frac{\partial f}{\partial x_i}\bigl(x_1,\dots,x_m\bigr)\,\frac{d v_i(t)}{dt}. dtdf​=i=1∑m​∂xi​∂f​(x1​,…,xm​)dtdvi​(t)​.

这就是所求的 (a) 小问的答案。

(b) 具体函数与具体参数函数的例子

给定

m=2,f(x1,x2)=x12  +  x1x2  +  2x22,m = 2,\quad f(x_1,x_2) = x_1^2 \;+\; x_1 x_2 \;+\; 2 x_2^2, m=2,f(x1​,x2​)=x12​+x1​x2​+2x22​,

并且

v1(t)=sin⁡t,v2(t)=et.v_1(t) = \sin t,\quad v_2(t) = e^t. v1​(t)=sint,v2​(t)=et.

令 x1=v1(t)=sin⁡tx_1 = v_1(t) = \sin tx1​=v1​(t)=sint, x2=v2(t)=etx_2 = v_2(t) = e^tx2​=v2​(t)=et。要求 dfdt\tfrac{df}{dt}dtdf​。

  1. 先计算偏导数：

    ∂f∂x1=2x1+x2,\frac{\partial f}{\partial x_1} = 2 x_1 + x_2, ∂x1​∂f​=2x1​+x2​,

    ∂f∂x2=x1+4x2.\frac{\partial f}{\partial x_2} = x_1 + 4 x_2. ∂x2​∂f​=x1​+4x2​.

  2. 将 x1,x2x_1, x_2x1​,x2​ 替换成 sin⁡t,et\sin t, e^tsint,et：

    ∂f∂x1(sin⁡t,et)=2sin⁡t  +  et,\frac{\partial f}{\partial x_1}\Bigl(\sin t, e^t\Bigr) = 2\sin t \;+\; e^t, ∂x1​∂f​(sint,et)=2sint+et,

    ∂f∂x2(sin⁡t,et)=sin⁡t  +  4et.\frac{\partial f}{\partial x_2}\Bigl(\sin t, e^t\Bigr) = \sin t \;+\; 4e^t. ∂x2​∂f​(sint,et)=sint+4et.

  3. 还需要 dv1dt=cos⁡t\frac{dv_1}{dt} = \cos tdtdv1​​=cost 和 dv2dt=et\frac{dv_2}{dt} = e^tdtdv2​​=et。

  4. 由链式法则：

    dfdt=(2sin⁡t+et) cos⁡t  +  (sin⁡t+4et) et.\frac{df}{dt} = \bigl(2 \sin t + e^t\bigr)\,\cos t \;+\; \bigl(\sin t + 4 e^t\bigr)\,e^t. dtdf​=(2sint+et)cost+(sint+4et)et.

可根据需要再做一些展开，但通常保持上述形式即可。若展开则是

dfdt=2sin⁡tcos⁡t  +  etcos⁡t  +  sin⁡t et  +  4e2t.\frac{df}{dt} = 2\sin t \cos t \;+\; e^t \cos t \;+\; \sin t\,e^t \;+\; 4 e^{2t}. dtdf​=2sintcost+etcost+sintet+4e2t.

这就是所求的 dfdt\frac{df}{dt}dtdf​。

2. 一阶线性常微分方程

考虑微分方程

dydx  −  2x y  =  ex2.\frac{dy}{dx} \;-\; 2x\,y \;=\; e^{x^2}. dxdy​−2xy=ex2.

这是一个标准的一阶线性方程

dydx+p(x) y=q(x),\frac{dy}{dx} + p(x)\,y = q(x), dxdy​+p(x)y=q(x),

的形式，只是这里 p(x)=−2xp(x)=-2xp(x)=−2x，q(x)=ex2q(x)=e^{x^2}q(x)=ex2。
其解法通常用“积分因子” (integrating factor)：

  1. 积分因子

    μ(x)=exp⁡(∫−2x dx)=exp⁡(−x2).\mu(x) = \exp\Bigl(\int -2x \,dx\Bigr) = \exp\bigl(-x^2\bigr). μ(x)=exp(∫−2xdx)=exp(−x2).

  2. 将原方程左右同乘以 μ(x)\mu(x)μ(x) 得

    e−x2dydx  −  2x e−x2 y=e−x2 ex2=1.e^{-x^2} \frac{dy}{dx} \;-\; 2x\, e^{-x^2}\,y = e^{-x^2}\, e^{x^2} = 1. e−x2dxdy​−2xe−x2y=e−x2ex2=1.

    观察左侧可知它是

    ddx(y e−x2),\frac{d}{dx}\bigl(y\, e^{-x^2}\bigr), dxd​(ye−x2),

    因为

    ddx(y e−x2)=e−x2 dydx+y ddx(e−x2)=e−x2 dydx  −  2x e−x2 y.\frac{d}{dx}\bigl(y\, e^{-x^2}\bigr) = e^{-x^2}\,\frac{dy}{dx} + y \,\frac{d}{dx}\bigl(e^{-x^2}\bigr) = e^{-x^2}\,\frac{dy}{dx} \;-\;2x\, e^{-x^2}\,y. dxd​(ye−x2)=e−x2dxdy​+ydxd​(e−x2)=e−x2dxdy​−2xe−x2y.

    这正好与我们上面左侧相同。

  3. 因此积分化简为

    ddx(y e−x2)=1.\frac{d}{dx}\bigl(y\, e^{-x^2}\bigr) = 1. dxd​(ye−x2)=1.

    对 xxx 积分：

    y e−x2=∫1 dx=x+C,y\, e^{-x^2} = \int 1 \,dx = x + C, ye−x2=∫1dx=x+C,

    其中 CCC 为任意常数。

  4. 最后解出 yyy：

    y=ex2 (x+C).y = e^{x^2}\,(x + C). y=ex2(x+C).

这就是方程的通解。

3. 复积分 ∮Ccos⁡z(2z−π)3 dz\displaystyle \oint_C \frac{\cos z}{(2z - \pi)^3}\,dz∮C​(2z−π)3cosz​dz，其中 C:∣z∣=2C: |z|=2C:∣z∣=2

我们要计算

∮Ccos⁡z(2z−π)3 dz,\oint_C \frac{\cos z}{(2z-\pi)^3}\,dz, ∮C​(2z−π)3cosz​dz,

其中 CCC 是以原点为中心、半径 2 的圆 ∣z∣=2|z|=2∣z∣=2。

  1. 奇点及其位置
    分母 (2z−π)3=0(2z - \pi)^3 = 0(2z−π)3=0 给出奇点 z=π2z = \frac{\pi}{2}z=2π​。
    数值上 π≈3.14159\pi \approx 3.14159π≈3.14159，故 π2≈1.5708\frac{\pi}{2} \approx 1.57082π​≈1.5708，其绝对值约 1.57，小于 2。因此 z=π2z=\frac{\pi}{2}z=2π​ 在圆 ∣z∣=2|z|=2∣z∣=2 的内部。

  2. 计算方法
    该奇点是三阶极点 (third-order pole)。根据留数定理 (Residue Theorem)，若函数 f(z)f(z)f(z) 在 z0z_0z0​ 是一个极点且位于闭合曲线 CCC 内，则

    ∮Cf(z) dz=2πi ⋅ Res(f,z0).\oint_C f(z)\,dz = 2\pi i \,\cdot\, \mathrm{Res}\bigl(f, z_0\bigr). ∮C​f(z)dz=2πi⋅Res(f,z0​).

    这里要找

    Res ⁣(cos⁡z(2z−π)3, z=π2).\mathrm{Res}\!\Bigl(\frac{\cos z}{(2z-\pi)^3},\,z=\tfrac{\pi}{2}\Bigr). Res((2z−π)3cosz​,z=2π​).

  3. 留数的计算
    对于三阶极点，可以用“通用留数公式”：

    Res(f,z0)=12! lim⁡z→z0d2dz2[(z−z0)3 f(z)].\mathrm{Res}\bigl(f,z_0\bigr) = \frac{1}{2!}\,\lim_{z \to z_0} \frac{d^2}{dz^2}\Bigl[(z - z_0)^3 \,f(z)\Bigr]. Res(f,z0​)=2!1​z→z0​lim​dz2d2​[(z−z0​)3f(z)].

    在此处 z0=π2z_0 = \frac{\pi}{2}z0​=2π​。先观察

    (z−π2)3 cos⁡z(2z−π)3.(z - \tfrac{\pi}{2})^3 \,\frac{\cos z}{(2z-\pi)^3}. (z−2π​)3(2z−π)3cosz​.

    注意到

    2z−π=2(z−π2),2z - \pi = 2\bigl(z - \tfrac{\pi}{2}\bigr), 2z−π=2(z−2π​),

    因而

    (z−π2)3 1(2z−π)3=(z−π2)3 1(2(z−π2))3=18.(z - \tfrac{\pi}{2})^3 \,\frac{1}{(2z-\pi)^3} = (z-\tfrac{\pi}{2})^3 \,\frac{1}{\bigl(2(z-\tfrac{\pi}{2})\bigr)^3} = \frac{1}{8}. (z−2π​)3(2z−π)31​=(z−2π​)3(2(z−2π​))31​=81​.

    于是

    (z−π2)3 cos⁡z(2z−π)3=cos⁡z8.(z - \tfrac{\pi}{2})^3 \,\frac{\cos z}{(2z-\pi)^3} = \frac{\cos z}{8}. (z−2π​)3(2z−π)3cosz​=8cosz​.

    根据留数公式要取它对 zzz 的二阶导数并在 z→π2z\to \frac{\pi}{2}z→2π​ 处极限，然后再乘 12!\tfrac{1}{2!}2!1​。

    计算二阶导数：

    ddz(cos⁡z8)=−sin⁡z8,d2dz2(cos⁡z8)=−cos⁡z8.\frac{d}{dz}\Bigl(\tfrac{\cos z}{8}\Bigr) = \tfrac{-\sin z}{8}, \qquad \frac{d^2}{dz^2}\Bigl(\tfrac{\cos z}{8}\Bigr) = \tfrac{-\cos z}{8}. dzd​(8cosz​)=8−sinz​,dz2d2​(8cosz​)=8−cosz​.

    在 z=π2z=\frac{\pi}{2}z=2π​ 代入可得

    −cos⁡z8∣z=π2=−cos⁡(π2)8=−08=0.\left.\frac{-\cos z}{8}\right|_{z=\tfrac{\pi}{2}} = -\frac{\cos(\tfrac{\pi}{2})}{8} = -\frac{0}{8} = 0. 8−cosz​​z=2π​​=−8cos(2π​)​=−80​=0.

    所以该二阶导数在 z=π2z=\frac{\pi}{2}z=2π​ 的值是 0。由此可见

    Res(cos⁡z(2z−π)3, z=π2)=12! × 0=0.\mathrm{Res}\Bigl(\frac{\cos z}{(2z-\pi)^3},\, z=\tfrac{\pi}{2}\Bigr) = \frac{1}{2!}\,\times\,0 = 0. Res((2z−π)3cosz​,z=2π​)=2!1​×0=0.

  4. 由留数定理得出积分值
    既然留数为 0，那么

    ∮Ccos⁡z(2z−π)3 dz=2πi × 0=0.\oint_C \frac{\cos z}{(2z - \pi)^3}\,dz = 2\pi i \,\times\, 0 = 0. ∮C​(2z−π)3cosz​dz=2πi×0=0.

因此，该复积分等于 0。

最终答案

  1. (a)

dfdt=∑i=1m∂f∂xi dvidt.\frac{df}{dt} = \sum_{i=1}^{m} \frac{\partial f}{\partial x_i}\,\frac{d v_i}{dt}. dtdf​=i=1∑m​∂xi​∂f​dtdvi​​.

(b) 当 m=2, f(x1,x2)=x12+x1x2+2x22,  v1(t)=sin⁡t,  v2(t)=etm=2,\,f(x_1,x_2)=x_1^2 + x_1 x_2 + 2x_2^2,\; v_1(t)=\sin t,\;v_2(t)=e^tm=2,f(x1​,x2​)=x12​+x1​x2​+2x22​,v1​(t)=sint,v2​(t)=et 时，

dfdt=(2sin⁡t+et)cos⁡t  +  (sin⁡t+4et)et.\frac{df}{dt} = \bigl(2\sin t + e^t\bigr)\cos t \;+\; \bigl(\sin t + 4e^t\bigr) e^t. dtdf​=(2sint+et)cost+(sint+4et)et.

的通解为

y=ex2 (x+C),y = e^{x^2}\,(x + C), y=ex2(x+C),
