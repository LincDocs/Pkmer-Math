概率密度例题

LincDocs2025年1月30日大约 2 分钟数学例题

確率密度関数 p(x)p(x)p(x) を持つ確率変数 XXX に対して，モーメント母関数 M(θ)M(\theta)M(θ) は、変数 θ\thetaθ を用いて次式で定義されます：

M(θ)=E[exp⁡(θX)]=∫−∞∞exp⁡(θx)p(x) dxM(\theta) = E[\exp(\theta X)] = \int_{-\infty}^{\infty} \exp(\theta x) p(x) \, dx M(θ)=E[exp(θX)]=∫−∞∞​exp(θx)p(x)dx

また，N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2) は，平均 μ\muμ，分散 σ2\sigma^2σ2 の正規分布を表し，その確率密度関数 pN(x)p_N(x)pN​(x) は次式で定義されます：

pN(x)=12πσ2exp⁡(−(x−μ)22σ2)p_N(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x - \mu)^2}{2\sigma^2}\right) pN​(x)=2πσ2​1​exp(−2σ2(x−μ)2​)

(1) N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2) のモーメント母関数を MN(θ)M_N(\theta)MN​(θ) としたとき、次式を導出します：

MN(θ)=exp⁡(μθ+σ22θ2)M_N(\theta) = \exp\left(\mu \theta + \frac{\sigma^2}{2} \theta^2\right) MN​(θ)=exp(μθ+2σ2​θ2)

まず、MN(θ)M_N(\theta)MN​(θ) を定義式に従って計算します：

MN(θ)=E[exp⁡(θX)]=∫−∞∞exp⁡(θx)pN(x) dxM_N(\theta) = E[\exp(\theta X)] = \int_{-\infty}^{\infty} \exp(\theta x) p_N(x) \, dx MN​(θ)=E[exp(θX)]=∫−∞∞​exp(θx)pN​(x)dx

ここで、pN(x)p_N(x)pN​(x) を代入します：

MN(θ)=∫−∞∞exp⁡(θx)12πσ2exp⁡(−(x−μ)22σ2) dxM_N(\theta) = \int_{-\infty}^{\infty} \exp(\theta x) \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x - \mu)^2}{2\sigma^2}\right) \, dx MN​(θ)=∫−∞∞​exp(θx)2πσ2​1​exp(−2σ2(x−μ)2​)dx

指数関数の部分をまとめます：

MN(θ)=12πσ2∫−∞∞exp⁡(θx−(x−μ)22σ2) dxM_N(\theta) = \frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} \exp\left(\theta x - \frac{(x - \mu)^2}{2\sigma^2}\right) \, dx MN​(θ)=2πσ2​1​∫−∞∞​exp(θx−2σ2(x−μ)2​)dx

ここで、θx−(x−μ)22σ2\theta x - \frac{(x - \mu)^2}{2\sigma^2}θx−2σ2(x−μ)2​ を簡略化します。まず、分母を共通化します：

θx−(x−μ)22σ2=θx−x2−2μx+μ22σ2=θx−x22σ2+μxσ2−μ22σ2\theta x - \frac{(x - \mu)^2}{2\sigma^2} = \theta x - \frac{x^2 - 2\mu x + \mu^2}{2\sigma^2} = \theta x - \frac{x^2}{2\sigma^2} + \frac{\mu x}{\sigma^2} - \frac{\mu^2}{2\sigma^2} θx−2σ2(x−μ)2​=θx−2σ2x2−2μx+μ2​=θx−2σ2x2​+σ2μx​−2σ2μ2​

θx\theta xθx と μxσ2\frac{\mu x}{\sigma^2}σ2μx​ をまとめます：

θx+μxσ2=θσ2+μσ2x\theta x + \frac{\mu x}{\sigma^2} = \frac{\theta \sigma^2 + \mu}{\sigma^2} x θx+σ2μx​=σ2θσ2+μ​x

したがって、式全体は次のようになります：

exp⁡(θx−(x−μ)22σ2)=exp⁡(−x22σ2+θσ2+μσ2x−μ22σ2)\exp\left(\theta x - \frac{(x - \mu)^2}{2\sigma^2}\right) = \exp\left(-\frac{x^2}{2\sigma^2} + \frac{\theta \sigma^2 + \mu}{\sigma^2} x - \frac{\mu^2}{2\sigma^2}\right) exp(θx−2σ2(x−μ)2​)=exp(−2σ2x2​+σ2θσ2+μ​x−2σ2μ2​)

ここで、補完平方（ completing the square）を使います：

−x22σ2+θσ2+μσ2x=−12σ2(x2−2(θσ2+μ)x)=−12σ2(x−(θσ2+μ))2+(θσ2+μ)22σ2-\frac{x^2}{2\sigma^2} + \frac{\theta \sigma^2 + \mu}{\sigma^2} x = -\frac{1}{2\sigma^2} \left( x^2 - 2(\theta \sigma^2 + \mu)x \right) = -\frac{1}{2\sigma^2} \left( x - (\theta \sigma^2 + \mu) \right)^2 + \frac{(\theta \sigma^2 + \mu)^2}{2\sigma^2} −2σ2x2​+σ2θσ2+μ​x=−2σ21​(x2−2(θσ2+μ)x)=−2σ21​(x−(θσ2+μ))2+2σ2(θσ2+μ)2​

したがって、

exp⁡(θx−(x−μ)22σ2)=exp⁡(−12σ2(x−(θσ2+μ))2+(θσ2+μ)22σ2−μ22σ2)\exp\left(\theta x - \frac{(x - \mu)^2}{2\sigma^2}\right) = \exp\left(-\frac{1}{2\sigma^2} \left( x - (\theta \sigma^2 + \mu) \right)^2 + \frac{(\theta \sigma^2 + \mu)^2}{2\sigma^2} - \frac{\mu^2}{2\sigma^2}\right) exp(θx−2σ2(x−μ)2​)=exp(−2σ21​(x−(θσ2+μ))2+2σ2(θσ2+μ)2​−2σ2μ2​)

この式を MN(θ)M_N(\theta)MN​(θ) に代入します：

MN(θ)=12πσ2∫−∞∞exp⁡(−12σ2(x−(θσ2+μ))2)exp⁡((θσ2+μ)22σ2−μ22σ2) dxM_N(\theta) = \frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} \exp\left(-\frac{1}{2\sigma^2} \left( x - (\theta \sigma^2 + \mu) \right)^2\right) \exp\left(\frac{(\theta \sigma^2 + \mu)^2}{2\sigma^2} - \frac{\mu^2}{2\sigma^2}\right) \, dx MN​(θ)=2πσ2​1​∫−∞∞​exp(−2σ21​(x−(θσ2+μ))2)exp(2σ2(θσ2+μ)2​−2σ2μ2​)dx

ここで、定数部分を積分の外に出します：

MN(θ)=exp⁡((θσ2+μ)22σ2−μ22σ2)12πσ2∫−∞∞exp⁡(−12σ2(x−(θσ2+μ))2) dxM_N(\theta) = \exp\left(\frac{(\theta \sigma^2 + \mu)^2}{2\sigma^2} - \frac{\mu^2}{2\sigma^2}\right) \frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} \exp\left(-\frac{1}{2\sigma^2} \left( x - (\theta \sigma^2 + \mu) \right)^2\right) \, dx MN​(θ)=exp(2σ2(θσ2+μ)2​−2σ2μ2​)2πσ2​1​∫−∞∞​exp(−2σ21​(x−(θσ2+μ))2)dx

最後の積分部分は正規分布の確率密度関数の積分なので、1 になります：

MN(θ)=exp⁡((θσ2+μ)22σ2−μ22σ2)M_N(\theta) = \exp\left(\frac{(\theta \sigma^2 + \mu)^2}{2\sigma^2} - \frac{\mu^2}{2\sigma^2}\right) MN​(θ)=exp(2σ2(θσ2+μ)2​−2σ2μ2​)

式を整理します：

MN(θ)=exp⁡(θ2σ4+2θσ2μ+μ22σ2−μ22σ2)M_N(\theta) = \exp\left(\frac{\theta^2 \sigma^4 + 2\theta \sigma^2 \mu + \mu^2}{2\sigma^2} - \frac{\mu^2}{2\sigma^2}\right) MN​(θ)=exp(2σ2θ2σ4+2θσ2μ+μ2​−2σ2μ2​)

μ2\mu^2μ2 がキャンセルされます：

MN(θ)=exp⁡(θ2σ4+2θσ2μ2σ2)M_N(\theta) = \exp\left(\frac{\theta^2 \sigma^4 + 2\theta \sigma^2 \mu}{2\sigma^2}\right) MN​(θ)=exp(2σ2θ2σ4+2θσ2μ​)

最後に：

MN(θ)=exp⁡(μθ+σ22θ2)M_N(\theta) = \exp\left(\mu \theta + \frac{\sigma^2}{2} \theta^2\right) MN​(θ)=exp(μθ+2σ2​θ2)

これで、N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2) のモーメント母関数 MN(θ)M_N(\theta)MN​(θ) が導出されました。
