$(F, +)$ 是一个交换群
- Closure![[Closure]]：对于任意 $a, b \in F$，$a + b \in F$。
- 结合性：对于任意 $a, b, c \in F$，$(a + b) + c = a + (b + c)$。
- 交换性：对于任意 $a, b \in F$，$a + b = b + a$。
- 存在加法单位元：存在 $0 \in F$，使得对于任意 $a \in F$，$a + 0 = a$。
- 存在加法逆元：对于任意 $a \in F$，存在 $-a \in F$，使得 $a + (-a) = 0$。
